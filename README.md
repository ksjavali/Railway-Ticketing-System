## Getting started:
### Clone the repository:
```
git clone https://github.ncsu.edu/ksjavali/CSC-517-Program2
```

### Go to the directory:
```
cd CSC-517_Program2
```

### Install required Gems:
```
bundle install
```
### Run database migration on your system:
```
rails db:migrate
```
### Run seed for the setting up required data:
```
rails db:seed
```
### Run the rails server:
```
rails server
```
